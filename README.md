# 線上課程預約系統 (Java Swing)

這是一個基於 Java Swing 開發的教學平台，目前已完成「學生端」的所有功能。

## 🌟 學生端核心功能
* **點數系統**：儲值（1:10）與自動扣點機制。
* **預約課程**：支援購物車多堂預約、防止重複預約鎖定。
* **個人課表**：查看預約紀錄與 Zoom 視訊連結。
* **評價系統**：完課後可對老師進行 1-5 星評分。

## 🛠 技術架構
* **語言**: Java 17
* **資料庫**: MySQL 8.0
* **設計模式**: 三層式架構 (DAO / Service / UI)
* **防護機制**: 使用 PreparedStatement 防止 SQL 注入。
